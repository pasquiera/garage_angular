À propos d’Herocar

Herocar est un projet réalisé par moi-même, Arnaud Pasquier, étudiant à l’université d'Orléans, dont le but est de démontrer mes compétences techniques en Angular. Le site a été développé avec Angular 12 CLI et possède une base de données Firebase.

Herocar est un site d'enchères automobiles. Il permet à un utilisateur disposant d’un compte de mettre en vente un ou plusieurs véhicules et d’enchérir sur les annonces d’autres utilisateurs. Le site propose aussi d’autres fonctionnalités telles que la création d’un compte, la modification des paramètres utilisateur, la modification des annonces après upload ou encore un espace commentaire propre à chaque annonce.

Vous trouverez un détail des fonctionnalités, page par page dans la section “Quelles sont les fonctionnalités du site?” de la FAQ. Vous trouverez le code source complet du site à l’adresse suivante : https://github.com/pasquiera/garage_angular.git

Pour toutes questions, n'hésitez pas à me contacter sur mon adresse mail : pasquier.arn@gmail.com

Quelles sont les fonctionnalités du site?

Le site est divisé en deux parties, une partie accessible à tous les visiteurs et une partie nécessitant la création d'un compte.

Partie accessible à tous:

/auctions

  • Cette page affiche l'ensemble des enchères en cours.
  • Les annonces peuvent être filtrées selon le type du véhicule, la date ou le prix.
  • Afin d’éviter d'éventuels ralentissements, les enchères sont récupérées depuis la base de données par groupe de 4 (lazy loading).
  • Le bouton "charger plus" permet d'afficher les 4 enchères suivantes.

/car/[:id]

  • Cette page est générée automatiquement pour chaque annonce, elle affiche les détails d'un véhicule.
  • Elle dispose d'une section “Enchérir” permettant aux utilisateurs disposant d'un compte d'enchérir sur une annonce.
  • Elle dispose d'une section commentaire permettant aux utilisateurs de poster ou répondre à un commentaire.

/sell

  • Cette page présente les avantages de Herocar.
  • Le bouton "Commencer" redirige vers la fenêtre d'authentification.
  • La section "Actuellement en vente sur Herocar" est mise à jour en fonction des enchères en cours.

/faq

  • Cette page dispose d'un formulaire de contact fonctionnel.
Partie nécessitant la création d'un compte (accès restreint par Route Guards):

/create

  • Cette page dispose d'un formulaire permettant de mettre en vente un véhicule.
  • Le formulaire dispose d'une boîte d'upload permettant d'ajouter des images. Un système de drag and drop permet de changer l'ordre dans lequel les images de l'annonce apparaissent.

/edit/[:id]

  • Cette page permet d'éditer toutes les informations d'une annonce.
  • Les informations sont récupérées depuis la base de données (Firebase) et modifiables via un reactive form.

/settings

  • Cette page permet à l'utilisateur de modifier ses informations personnelles.
  • En cas de changement d'adresse email, un mail de confirmation est envoyé à la nouvelle adresse.
  • La page dispose d'un module permettant de rogner la taille de l'image de profil lors de l’upload.

/bid

  • Cette page regroupe les annonces postées par l'utilisateur (Mes véhicules en vente) et les annonces sur lesquelles il a enchéri (Mes enchères en cours).
  • Chaque annonce postée par l'utilisateur est modifiable en cliquant sur l'icône edit ou supprimable en cliquant sur l'icône delete .
  • La section "Mes enchères en cours" est automatiquement mise à jour si l'utilisateur n'est plus en tête d'une enchère.
